# Notatki produktowe

Next Door App Builder może być wewnętrznym silnikiem do tworzenia mikroaplikacji w ekosystemie Next Door Studio.

Może zasilać:
- Przepiśnik,
- BriefBox,
- RepairFlow,
- osobiste planery,
- katalogi,
- checklisty,
- organizery.

Najważniejsze ograniczenie: produkt musi pozostać prosty.
