# Wizja produktu

Next Door App Builder ma być kreatorem mikroaplikacji użytkowych.

Nie jest pełnym no-code builderem ani konkurencją dla Bubble, Replit czy FlutterFlow. To system składania prostych aplikacji z gotowych modułów.

Użytkownik wybiera typ aplikacji, pola danych, podstawowy układ i styl. System składa z tego gotową mikroaplikację.

Główna przewaga: prostota, gotowe zastosowania i brak technicznego przeciążenia.
